<p><span style="font-size: 14px; font-family: verdana;"><strong>Tooltip Overhaul</strong> is a complete visual refresh for Minecraft tooltips, completely replacing the vanilla tooltip system with a clearer layout. Every tooltip, more modern, more clearer, more sharper.</span></p>
<p>&nbsp;</p>
<p><span style="font-size: 14px; font-family: verdana;">Within the core utilities, this mod includes:</span></p>
<p><span style="font-size: 14px; font-family: verdana;">- <strong>Scrollable tooltips</strong>. When a tooltip is too tall and goes beyond the screen limits, you will be able to scroll through its content.<br>-<strong> Text wrapping</strong>. Automatically, when a tooltip is too wide and goes past the screen limits, this will be prevented by inserting line breaks where needed.<br>- <strong>Compatible </strong>with JEI, EMI, and ModernFix (and flerovium).<br>- The<strong> inner border</strong> style can be <strong>changed</strong>, with 3 possible types: Glint (<em>default</em>), Static, and Gradient.<br>- A new tooltip element has been added, called <strong>rating</strong>. When hovering over an item, just below its name a qualifier will appear, which by default shows its rarity. This is configurable and can display whatever you want.<br>- A wide variety of <strong>effects </strong>and <strong>frames </strong>to choose from. Frames can also be animated, and additional overlays can be added through resource packs.<br>- Any mod can add its own <strong>custom frames</strong>.<br>- <strong>3D rendering</strong> of hovered items. A second panel will appear when looking at certain items.</span></p>
<p>&nbsp;</p>
<p><em>By default, some vanilla items include frames. If players think it&rsquo;s a good idea, we can create custom frames for the most requested mods and add native support. Reach out!</em></p>

<h2><strong>Configuration</strong></h2>
<p><span style="font-family: verdana;"><span style="font-size: 14px;">There are <strong>two </strong>ways to configure custom tooltips. For global configurations, locate the configuration file at&nbsp;<strong>config/tooltipoverhaul.toml</strong>, it contains general configurability that will affect every single tooltip. For specific configurations, that affect specific items (and tags), follow this <a href="https://github.com/Xylonity/Tooltip-Overhaul/wiki/Configuration" target="_blank" rel="nofollow noopener"><strong>wiki</strong></a>.</span></span></p>
<p>&nbsp;</p>
